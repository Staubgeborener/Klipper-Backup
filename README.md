Klipper-Backup is a script for manual or automated GitHub backups. It's Lightweight, pragmatic and comfortable.

## Installation

### Download:
```shell
curl -fsSL get.klipperbackup.xyz | bash
```

### Installation/Configuration:
```shell
~/klipper-backup/install.sh
```
